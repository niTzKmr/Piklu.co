# Piklu.co

React/Vite storefront for Piklu.co gifts, product browsing, product detail pages, and WhatsApp-based checkout.

## Development

```bash
npm install
npm run dev
```

The dev and build scripts regenerate `src/data/products.js` from `catalog.csv` before starting.

## Catalog Workflow

Product data is maintained in `catalog.csv`. Do not edit `src/data/products.js` directly; it is generated by:

```bash
node scripts/parse-catalog.cjs
```

Product images should live under `public/Products/<Product-Folder>/`. Use root-relative image paths in the catalog, for example:

```text
/Products/Pixel-Frame-4x4/Frame4x4.png
```

To sync images from `public/Products` into the catalog:

```bash
npm run sync
```

New synced products may get placeholder prices, descriptions, tags, and metadata. Review `catalog.csv` after syncing.

## Checks

```bash
npm run lint
npm run build
```

## Checkout

Checkout opens WhatsApp with a prefilled order message. The merchant phone number is configured in `src/components/CheckoutDrawer.jsx`.
